# ngal 剧情编写教程

## 文件位置

`assets/dialog/dialogue.txt`

## 基础结构

```txt
[场景名]
命令1
命令2
...
```

游戏从 `[welcome]` 场景开始。

## 完整演示

```txt
[welcome]
music:title.mp3
img:NAS油条.png:3:50%
NAS油条:本项目由Rust语言开发\n按回车键继续:nas_intro.mp3
img:
input:请输入你的名字:name
NAS油条:好的，{name}，哪个游戏牛逼?:gamenb.mp3
choose:原神牛逼:ysnb|鸣朝牛逼:mcnb|终末地牛逼:zmd

[ysnb]
bg:91.png
img:鸣朝.png:3:35%
鸣朝:鸣朝才牛逼😡:mcnb.mp3
鸣朝:原神不牛逼🤓:ys_no_nb.mp3
原神:原神才牛逼🤓👍:ysnb.mp3
原神:鸣朝不牛逼😡:mc_no_nb.mp3
load:ytnb

[mcnb]
img:原神.png
原神:原神才牛逼🤓👍:ysnb.mp3
原神:鸣朝不牛逼😡:mc_no_nb.mp3
鸣朝:鸣朝才牛逼😡:mcnb.mp3
鸣朝:原神不牛逼🤓:ys_no_nb.mp3

end

img:我.png
{name}:😋他们产的片才牛逼😋:ysmcnb.mp3
bg:
NAS油条:游戏结束
end
```

## 命令详解

### 对话与旁白

| 命令 | 格式 | 说明 |
|------|--------|--------|
| 对话 | `说话人:文本` | 仅显示文字 |
| 对话 |`说话人:文本:语音.mp3` | 语音文件放在 assets/voices/ 目录 |
| 旁白 | `文本内容` | 无说话人时显示为旁白 |

### 变量

| 命令 | 格式 | 说明 |
|------|--------|--------|
| 变量输入 | `input:提示语:变量名` | 用户输入内容存为变量 |
| 变量使用 | `{变量名}` | 在文本中替换为变量值 |

### 图片与背景

| 命令 | 格式 | 说明 |
|------|--------|--------|
| 立绘 | `img:文件名.png` | 放 assets/portraits/ 目录 |
| 立绘位置缩放 | `img:文件:1\|2\|3:百分比%` | 1=左 2=中 3=右 |
| 清除立绘 | `img:` | 留空即清除当前立绘 |
| 背景图 | `bg:文件名.png` | 拉伸填满整个背景 |
| 清除背景 | `bg:` | 留空即清除背景图 |

### 音频

| 命令 | 格式 | 说明 |
|------|--------|--------|
| 背景音乐 | `music:文件名.mp3` | 放 assets/music/ 目录 |
| 停止音乐 | `music:` | 留空停止当前播放的音乐 |

### 流程控制

| 命令 | 格式 | 说明 |
|------|--------|--------|
| 选项分支 | `choose:选项1:场景1\|选项2:场景2` | 用竖线分隔多个选项 |
| 跳转 | `load:场景名` | 跳转到指定场景 |
| 结束游戏 | `end` | 返回主菜单 |

## 路径说明

- 图片、语音、音乐文件均放在 `assets/` 下的对应子目录
- 文件名支持中文和空格

## 注意事项

- 文本中的 `\n` 表示换行
- 命令和参数大小写敏感
- 场景名只能用字母、数字、下划线
- 确保文件编码为 UTF-8（无 BOM）
